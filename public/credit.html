Ultraviolet by Titanium Network Ultraviolet is an advanced open-source web proxy developed by the Titanium Network, designed to help users bypass internet censorship. This powerful software forms the foundation of Fish Proxy, enabling users to access websites and navigate the web freely. Amazon EC2 Amazon EC2 provides the virtual server infrastructure that allows Fish Proxy to operate reliably online. By running Linux virtual machines in the cloud, EC2 enables Ultraviolet and related services to function efficiently and securely.
