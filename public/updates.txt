Build Classifications

ALPHA
  Early-stage builds prior to beta. These versions are highly unstable, intended for internal testing, and are almost never released publicly.

BETA
  Pre-release builds that may be functional but are not yet finalized. These versions may be suitable for limited release but can still contain significant issues.

STABLE
  Fully released builds considered complete. These versions are generally stable, include meaningful improvements from prior releases, and are intended for public use.
  *Note: “Stable” does not guarantee a completely bug-free experience.*

Version History

Fish Proxy v1.0 — BETA

* Introduced default starry background
* Added homepage banner and search bar
* Implemented dual proxy support for Ultraviolet and Scramjet *(not fully stable)*
* Included browser console functionality within websites

Fish Proxy v1.1 — STABLE

* Improved stability and speed of Ultraviolet (UV) and Scramjet (SJ) proxy switching
* Resolved lag issues during proxy transitions
* Added lazy loading and advanced caching for Scramjet to enhance performance during switching

Fish Proxy v2.0 — ALPHA

* Added custom background feature
* Users can upload images (JPEG and GIF formats supported)

Fish Proxy v2.1 — BETA

* Added quick links for Roblox
* Introduced a new games section with approximately 20 games
* Disabled Chromium-based browser recent search suggestions on the homepage and browser console
* Replaced the original Ultraviolet README with a custom README.md featuring improved visuals and updated project information
